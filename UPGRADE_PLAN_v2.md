# 安全生产管理系统升级方案 V2

## 参考目标
**远程网站**：http://222.133.13.78:8088/ （德州宏坤橡塑助剂有限公司 - 工业互联网平台）

## 一、远程网站结构分析

### 1.1 整体架构
- **技术栈**：Vue + Element UI + ECharts
- **布局模式**：
  - 导航页：全屏深色工业风驾驶舱
  - 功能页：顶部横向子菜单 + 左侧快捷导航(menus-box) + 内容区
- **字体**：Noto Sans SC

### 1.2 一级模块（14个）
1. **导航页** - 功能入口驾驶舱
2. **驾驶舱** - 数据大屏
3. **基础档案** - 人员/化学品/区域/三同时/设备/相关方/生产静态/安全生产档案/到期提醒
4. **安全办公** - 个人工作台/统计分析/会议与公告/文件管理/知识库/安全投入/目标责任书/变更管理/职业健康/员工考核
5. **教育培训** - 个人工作台/统计分析/资料库管理/培训设置/培训记录/年度学时考核
6. **双重预防** - 个人工作台/数据监控/工作推进/基础配置/包保责任人/日常排查/政府专项检查/隐患治理/运行评估/绩效评估/预警管理/建设成效评估
7. **作业管理** - 个人工作台/统计分析/基础配置/JSA分析/作业票预约/申请/现场办票/过程管理/归档/预警
8. **危险源监测** - 在线监测/实时监测管理/报警配置/报警管理/重大危险源档案/化学品档案
9. **相关方管理** - 个人工作台/统计分析/基础配置/相关方档案/考评奖惩
10. **风险管控** - 个人工作台/统计分析/风险研判/四色图评估/内部安全检查/随手拍/专家检查/隐患治理/三卡管理/动态风险
11. **事故应急** - 个人工作台/统计分析/基础配置/应急资源/应急预案/应急演练/事故管理/自然灾害预警
12. **电子巡检** - 个人工作台/统计分析/巡检内容配置/巡检点配置/巡检计划/巡检记录
13. **封闭化管理** - 个人工作台/基础信息/定位管理/聚集预警/访客管理
14. **视频IOT平台** - 监控平台/视频管理/AI管理
15. **上传模块**
16. **流程办理** - 待办流程/我发起的/我的抄送

### 1.3 设计特征
- **登录页**：蓝色渐变左侧+3D数据仪表盘插图，右侧白色圆角卡片
- **导航页**：深色背景+工厂实景底图，左侧用户栏，中间功能卡片网格，右侧天气面板
- **功能页**：白色背景，顶部蓝色横向导航条(el-menu--horizontal)，左侧menus-box快捷入口
- **表格**：proTable样式，Element UI标准表格

### 阶段二：功能扩展

6. 登录页风格统一
7. 数据看板增强

### 阶段三：细节优化
