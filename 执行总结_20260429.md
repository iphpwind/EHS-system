## 📋 用户需手动完成的操作

### 第一步：安装 Node.js

```powershell
# 以管理员身份运行
Start-Process "D:\phpstudy_pro\node-v20.11.1-x64.msi" -Verb RunAs
```

安装后验证：
```powershell
node -v
npm -v
```

### 第二步：切换 PHPStudy MySQL 版本到 8.0

1. 打开 PHPStudy 控制面板
2. 点击左侧 **数据库**
3. 点击 **MySQL** 下拉菜单
4. 选择 **MySQL8.0.12**
5. 点击 **重启服务**

切换后验证（在 PHPStudy 中点击 MySQL 后的「管理」按钮，或用命令行）：
```bash
"D:\phpstudy_pro\Extensions\MySQL8.0.12\bin\mysql.exe" -u root -proot -e "SELECT VERSION();"
```

### 第三步：导入数据库 Schema

```powershell
# 先创建数据库
"D:\phpstudy_pro\Extensions\MySQL8.0.12\bin\mysql.exe" -u root -proot -e "CREATE DATABASE IF NOT EXISTS safety_system DEFAULT CHARSET utf8mb4;"

# 导入 Schema
"D:\phpstudy_pro\Extensions\MySQL8.0.12\bin\mysql.exe" -u root -proot safety_system < "D:\phpstudy_pro\safety-system\backend\src\schema\init_db.sql"
```

### 第四步：安装前端依赖

```powershell
cd D:\phpstudy_pro\safety-system\frontend
npm install
```

### 第五步：安装后端依赖

```powershell
cd D:\phpstudy_pro\safety-system\backend
npm install
```

### 第六步：启动开发服务

**后端**（新开终端）：
```powershell
cd D:\phpstudy_pro\safety-system\backend
npm run dev
# 应显示：🚀 后端服务启动成功，监听端口 3001
```

**前端**（新开终端）：
```powershell
cd D:\phpstudy_pro\safety-system\frontend
npm run dev
# 应显示：Local: http://localhost:3000/
```
